# Assignment 5 – Module 6: Data Structures and Strings in Python

## 📌 Overview
This assignment demonstrates the use of **Python dictionaries**, **lists**, and **list slicing**. It focuses on storing and retrieving data using dictionaries and performing basic list operations such as slicing and reversing.

---

## 📂 Tasks

### Task 1: Create a Dictionary of Student Marks
- Creates a dictionary with student names as keys and their marks as values.
- Prompts the user to enter a student's name.
- Retrieves and displays the student's marks if the name exists.
- Displays an appropriate message if the student's name is not found in the dictionary.

### Task 2: Demonstrate List Slicing
- Creates a list of numbers from 1 to 10.
- Extracts the first five elements using list slicing.
- Reverses the extracted list.
- Displays both the original extracted list and the reversed list.
